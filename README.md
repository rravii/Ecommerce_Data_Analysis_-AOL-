## Codes for generating graph for BI assignmnet

Python version
~3.13.7

### Setup

- Optional: Create a virtual environment to isolate your package version for projects

```
python -m venv .venv
```

- To activate the Virtual environment on Windows

  - On your command prompt run:

    ```
    .venv\Scripts\activate.bat
    ```

  - OR On your Powershell run:

    ```
    .\.venv\Scripts\Activate.ps1
    ```

- To activate the Virtual environment on Mac or Linux run:

  ```
  source .venv/bin/activate
  ```

- To Deactivate your virtual environment run

  ```
  deactivate
  ```

- Install the python packages already in the `requirements.txt` file run:

```
pip install -r requirements.txt
```

### Run Code

- To Run Python code
  Type `python question1/chart-plot.py` in your terminal to run the chart for question 1, you can run other questions as follows.

- To Run SQL code
  Open the code in your DB UI editor in eithe DBeaver or DataGrip
